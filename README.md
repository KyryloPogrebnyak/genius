# Modern Bakery Website

A modern bakery website showcasing traditional baking with contemporary design and service quality.

## About

This is a Ukrainian bakery website that combines the best of traditional baking with modern efficiency. The bakery offers fresh baked goods ready in just 5 minutes while maintaining the authentic taste and natural ingredients of traditional recipes.

## Features

- Responsive HTML layout
- Multi-section navigation
- Chef profiles showcase
- Location integration with Google Maps
- Contact information and ordering system
- Traditional Ukrainian bakery branding

## Sections

- **Header**: Logo, navigation menu, and call-to-action button
- **Hero**: Main bakery introduction
- **Traditions**: Information about the bakery's approach and values
- **Chefs**: Profiles of head chefs Nastya and Vlad
- **Formats**: Different bakery format offerings (Walrus, Horseshoe, Handlebar)
- **Contacts**: Location map and contact details
- **Footer**: Contact information and links

## Technologies Used

- HTML5
- Semantic markup
- Google Maps integration
- Responsive design principles

## Getting Started

Simply open the `index.html` file in your web browser to view the website.
